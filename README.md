# NoteEditor

A minimal Note taking app with out-of-the-box markdown support.
Use hashtags, tab for blockquotes, double stars, underline and single quotations for the markdown inputs.

## Tech Stack
  #### Electron 
  #### React
  #### Typescript
  #### Tailwind
  #### Jotai


## Project Setup

### Install

```bash
$ npm install
```

### Development

```bash
$ npm start
```

### Build

```bash
$ npm run build
```
